面试多了, 能遇到一大堆声称能代替你面试的枪手公司的电话.
遇到这种电话, 恭喜你, 这说明你的个人信息已经被从不知道什么地方卖出去了, 而你对此无能为力.
在接下来的至少一两个月, 你都会断断续续接到代面公司打给你一些电话.

这种情况现在基本是无解的,
我们没法判断平台 (智联 / Boss 直聘) 上哪家公司是偷偷卖你个人信息或者就是代面公司本身,
只能对枪手打过来的电话稍微判断一下.

一般来说, 正经公司打电话过来, 格式会是: "XXX 您好, 我们是您在智联 / Boss 上投递过简历的 XXX 公司, 您的简历通过初筛, 现邀您参加下一步的面试".
正经公司会直接询问你什么时候有空参加下一步的面试, 如果有考试或者需要你自己准备的材料, 也会一并跟你确认.

代面公司的电话一般是完全不一样的, 首先第一句大概率是 "你好, 还在找工作吗":
对于你的名字, 对方很可能是不提的;
对于他所属的公司, 大概率也不会很乐意告诉你, 只有你主动确认对方身份, 对方才会支支吾吾告诉你一个公司名.
但是不论告诉你的是什么公司, 这个公司名大概率在智联 / Boss 上是查不到具体公司的.
要知道, 哪怕是外包公司, 在智联 / Boss 上大概率也是有一个能查到的公司实体存在的.

后续一般来说, 代面公司会询问你准备找什么方向的工作, 并且给你介绍他们公司可以提供什么岗位的工作.
这跟正经公司完全不同, 正经公司直接确认你能不能参加面试, 是 "公司需要你做什么", 代面公司更像是 "你需要对方做什么".

当你听到这种类型的电话之后, 你可以直接询问对方是不是代面公司.
目前来说, 代面公司的人听你问出这句话之后会直接亮明身份,
这一点跟外包公司不太一样, 很多外包公司是会通过一些有的没的话术尝试把自己包装得 "不那么外包".

确认是代面电话之后, 不用有什么挂电话的心理负担.
礼貌一点你可以直接说自己不用代面, 不用说太多.
对方一般来说听你说不用代面之后也不会跟你再扯太多内容, 大概率会直接挂电话.
